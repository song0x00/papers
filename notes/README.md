# notes/

Personal reference material, such as personal cheatsheets and usage applications for tools, technique breakdowns and personal research for certifications.



## Subdirectories

### [`tools/`](./tools/)
Usage notes, flags, and cheatsheets for tools in the workflow. Things like nmap, Burp, Wireshark, ffuf, etc.

### [`techniques/`](./techniques/)
Attack and defence technique breakdowns. Mapped to MITRE ATT&CK where relevant. Things like privilege escalation paths, lateral movement, persistence mechanisms.

### [`security-plus/`](./security-plus/)
Study notes and resources for the CompTIA Security+ certification.

### [`ceh/`](./ceh/)
Study notes and resources for the Certified Ethical Hacker (CEH) exam.
