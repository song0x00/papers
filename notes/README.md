# notes/

Running reference notes — certification study material, tool usage, and technique breakdowns. Less formal than whitepapers; more useful day-to-day.

---

## Subdirectories

### [`tools/`](./tools/)
Usage notes, flags, and cheatsheets for tools in the workflow. Things like nmap, Burp, Wireshark, ffuf, etc. — the kind of stuff that's faster to look up here than re-Google every time.

### [`techniques/`](./techniques/)
Attack and defence technique breakdowns. Mapped to MITRE ATT&CK where relevant. Good place for notes on things like privilege escalation paths, lateral movement, persistence mechanisms.

### [`security-plus/`](./security-plus/)
Study notes and resources for the CompTIA Security+ certification.

### [`ceh/`](./ceh/)
Study notes and resources for the Certified Ethical Hacker (CEH) exam.

---

These are living documents — expect them to be updated, expanded, and occasionally corrected as understanding improves.
